# Mary - Full-Stack Developer Portfolio

A modern, responsive portfolio showcasing full-stack development skills with Django, HTML, CSS, JavaScript, and MySQL.

## Features

- **Dark/Light Theme Toggle**: Switch between themes with persistent preference.
- **Responsive Design**: Optimized for desktop and mobile.
- **Interactive Elements**: Smooth scrolling, animations, and hover effects.
- **Contact Form**: Functional form using EmailJS for sending messages.
- **Project Showcase**: Highlighting key projects like the Boutique Inventory System.
- **Skills & Experience**: Visual representation of technical skills and professional background.

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript
- **Icons**: FontAwesome
- **Fonts**: Google Fonts (Montserrat, Roboto)
- **Email Service**: EmailJS
- **Deployment**: Ready for GitHub Pages, Netlify, or Vercel

## Setup

1. Clone the repository
2. Open `portfolio.html` in your browser
3. For contact form: Configure EmailJS with your credentials in `script.js`

## Sections

- **Home**: Hero section with introduction
- **About**: Personal information and photo
- **Experience**: Professional background
- **Skills**: Technical skills with progress bars
- **Projects**: Showcase of key projects
- **Testimonials**: Client/peer feedback
- **Contact**: Functional contact form

## Customization

- Update personal information in `portfolio.html`
- Modify colors in `style.css` CSS variables
- Add your projects and images
- Configure EmailJS for contact form
